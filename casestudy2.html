Holiday Soundboard Case Study

Overview:

The Holiday Soundboard was built to create a variety of holiday sound bites by using a
javascript drumkit. I also included an extra webpage for kid sounds that a parent may utilize.
Currently this only works on a computer and not on mobile.

Purpose & Context:

I was looking to create a new project that was simple and easy enough to digest for viewers.
I also was looking to learn new features I hadn't used before. This was my first time using
a drumkit with colored animation. After building up the Halloween sample page I realized I
could use this type of website to be used as a Streamdeck replacement that a streamer or
content creator could use.

Objective

The objective was to build a fully functional website that could be used as a simple Streamdeck
replacement. A Streamdeck being a piece of hardware that streamers push to play certain sound
clips or video clips while they are streaming. Therefore each button that you push that is on the
soundboard will play a sound bite. In this case each profile is based on different holidays
and a kid friendly page.

Duration:

This project took 1-2 weeks for full completion. This includes a few styling adjustments
such as editing the colors of the animations behind the keys and proper text sizing.
I also had to search for sound clips, edit them down, and then host the site online to use.

Credit:

Role: Lead Developer

"The How"

-An index.html file was created for our common webpage information. Initially only one soundboard
page was going to be created. Instead of creating multiple sites, it made more sense to create
multiple webpages/profiles.

-The heart of this project is matching the keys on the keyboard with the proper audio. To do this
we needed to create a working script to indicate that when I hit a data-key (A) that the matching
audio plays once it is hit ("Christmas Spirit" on Christmas).

-After everything was implemented and working properly it just needed to be spiced up a bit.
In styles.css I chose the colors for each webpage. The holiday text, the menubar colors change,
and the animation colors around the keys change.

"Challenges"

The major challenges for this project was having to define each holiday characteristics with CSS.
I also had to decide to find shorter audio clips (or edit them down) because I couldn't find a fix for
double clicking to stop audio. I also know this does not work on mobile without a keypad. These are
a few things I will try to figure out in the future.
